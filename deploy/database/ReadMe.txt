Steps to CONSTRUCT and DEPLOY the database:

- From a command prompt @ '/DomainDrivenDesignSample/construct/DDDSampleDb', run 'gradle distZip' command.
- From a command prompt @ '/DomainDrivenDesignSample/deploy/DDDSampleDb', run 'gradle deployNew'.

Other tasks include:

- From the same prompt, run 'gradle dropAll'.  This will drop all existing tables in the schema that was created above.
